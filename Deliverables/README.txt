SANTORINI README (AM005)

REQUIREMENTS : 
Java 13 (or superior) and JavaFX 11.0.2 (or superior) for the GUI. 
Make sure you download the correct jar for your OS.

SERVER:
Run the jar from the terminal using: java -jar path of the server jar socket port number

CLI: 
If you are using Windows, in order to correctly display ANSI Colors download an appropriate terminal (we suggest using ConEmu, you can download it from here: https://www.fosshub.com/ConEmu.html
Windows, MacOS: run the jar from the terminal using: java -jar path of the client jar server ip server socket port number

GUI: 
Windows, MacOS: run the jar from the terminal using: java-jar path of the client jar 
If you are using Windows, you can also launch the jar double-clicking on it.

Note: If you started your server locally, insert 127.0.0.1 as IP
